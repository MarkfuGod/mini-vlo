# Project Documents

Academic and planning materials for the Semantic-Motion / Mini-VLO project.
Kept in the private repository; not required to run the code.

| File | Description |
| --- | --- |
| `Master_project.pdf` | Master project brief |
| `Semantic-Motion_Proposal.pdf` | Project proposal |
| `Semantic-Motion_Interim_Report.docx` | Interim report |
| `PROJECT_REPORT_OUTLINE.md` | Final report writing outline (~15k words) |

Code, configs, tests, and experiment JSON live in the repository root.
Gold annotation packets are under `data/gold/` (candidate packets until adjudicated).
Large LIBERO raw/processed videos stay local under `data/libero_goal/` and are gitignored.
